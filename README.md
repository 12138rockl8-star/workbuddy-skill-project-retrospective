# 项目复盘报告生成器

基于 5 份优秀复盘文档（覆盖月度/季度/年度/事件/项目结案 5 种类型）的深度分析提炼，帮你从零到一产出专业复盘报告。

## 怎么用

说「项目复盘」或「写复盘」即可触发。Skill 会先问你项目信息、客户关注板块，给你一份定制化材料清单，你把材料喂给它，它就按「肯定成绩 → 分析问题 → 优化方案 → 展望未来」的逻辑线生成报告，最终输出为格式精美的钉钉文档。

## 核心能力

- **5 种复盘类型**：月度 / 季度 / 年度 / 项目结案 / 事件复盘
- **12 个客户关注板块**：内容运营 / 投流效果 / 品牌声量 / 营销活动 / 竞品分析 / 达人合作 / 品牌策略等
- **数据驱动 + 归因分析 + Learning 提炼 + 前后对比**，不写空话
- **自动生成钉钉文档**，分栏美化，交付链接

## 适合谁

代运营团队、品牌营销项目组、需要向客户交付复盘报告的 PM 和编导。
