# WorkBuddy Skill - 项目复盘报告生成器（v2.0）

基于 5 份优秀复盘文档的深度分析，提炼出通用的复盘逻辑、结构和语言风格，引导用户完成从信息收集到报告产出的全流程，最终生成**格式精美、数据可视化的钉钉文档**。

## v2.0 更新内容

- **新增蚂蚁森林季度复盘视觉风格**：渐变卡片、绿色数据高亮、TOP 案例卡片、引用块总结
- **强制图表可视化**：核心 KPI 达成率、内容占比、趋势对比、用户画像等数据必须转化为图表
- **更丰富的钉钉文档格式化**：支持彩色文字、HTML 卡片、图片+数据组合
- **新增 `references/visual-formatting.md`**：完整的视觉排版与图表可视化指南

## Skill 功能

### 三阶段流程

1. **信息收集**：询问项目名称/类型/周期 → 客户关注板块（12 个板块可选） → 生成定制化材料清单
2. **报告生成**：按复盘逻辑线（肯定成绩 → 分析问题 → 优化方案 → 展望未来）和六段式结构生成
3. **钉钉输出**：通过 `dws doc create` 创建格式精美的钉钉文档，交付链接给用户

### 支持的复盘类型

- 月度复盘
- 季度复盘
- 年度复盘
- 项目结案
- 事件复盘

### 客户关注板块（12 个）

内容运营 / 投流效果 / 社群运营 / 品牌声量 / 转化效果 / 客户满意度 / 粉丝增长 / 直播运营 / 营销活动 / 达人合作 / 竞品分析 / 品牌策略

## 文件结构

```
project-retrospective/
├── SKILL.md                          # 核心技能文件（触发词、交互流程、生成逻辑）
└── references/
    ├── analysis.md                   # 5 份复盘报告的深度分析与模式提炼
    ├── template.md                   # 完整复盘报告模板（含占位符）
    ├── dingtalk-formatting.md        # 钉钉文档格式化与美化指南
    └── visual-formatting.md          # v2.0 视觉排版与图表可视化指南
```

## 安装方式

### 方式一：直接拷贝

将 `project-retrospective` 文件夹拷贝到 WorkBuddy 的 skills 目录：

- **Windows**: `C:\ProgramData\WorkBuddy\users\<你的用户ID>\.workbuddy\skills\`
- **macOS/Linux**: `~/.workbuddy/skills/`

### 方式二：Git Clone

```bash
git clone https://github.com/12138rockl8-star/workbuddy-skill-project-retrospective.git
# 然后将 project-retrospective 文件夹拷贝到 skills 目录
```

## 使用方式

在 WorkBuddy 对话中输入以下任意触发词即可激活：

- "项目复盘"
- "复盘报告"
- "项目总结"
- "做复盘" / "写复盘"
- "月度/季度/年度复盘"

## 依赖

- **钉钉连接器**：用于最终输出钉钉文档（`dws doc create`）
- **腾讯文档连接器**：如需读取腾讯文档中的参考材料
- **Python 图表库（可选）**：如需生成 matplotlib/plotly/pyecharts 图表
